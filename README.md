## Getting Started

- install flutter SDK and Android SDK
  or see [getting started](https://flutter.dev/docs/get-started/install)
- install flutter/dart plugins in your IDE
- Clone the project
- run :
`flutter pub get`
- then :
`flutter run`
	or
run it directly with your IDE
